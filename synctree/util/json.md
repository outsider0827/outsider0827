### ● 결과

```text
{

  }
}
```

![](../../.gitbook/assets/image%20%28182%29.png)

### ● 예문

![](../../.gitbook/assets/image%20%28385%29.png)

![](../../.gitbook/assets/image%20%28456%29.png)

### ● 결과

```text
{
  "result": {
    "jsonDecode": {
      "test1": "test1",
      "test2": "test2"
    }
  }
}
```
